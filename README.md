# CLOS // Tools & Resources
**Campus Legends of Saturday — Internal Tools Repository**

This repository houses the web tools built for CLOS (Campus Legends of Saturday), a 32-coach online college football dynasty league built around EA Sports College Football. All tools are self-contained HTML files — no build process, no dependencies, just open and use.

---

## Tools

### Advance Tool
**`/advance-tool/clos_advance_tool.html`**

Handles everything needed on advance day. Generates the weekly advance post, Game of the Week article, GOTW graphic prompt, and Top 25 poll — all formatted and ready to copy. Syncs coach and Discord data live from the Team Info Sheet.

→ [View Advance Tool README](advance-tool/README.md)

---

### Wheel of Fortune
**`/wheel-of-fortune/clos_wheel_of_fortune.html`**

Powers the live Wheel of Fortune coaching carousel show. Coaches spin for job offers by tier, tiebreakers resolve automatically on screen, and the full session runs from setup through results. Pulls eligible coaches and occupied schools automatically from the Form Response Sheet and Team Info Sheet.

→ [View Wheel of Fortune README](wheel-of-fortune/README.md)

---

## How These Tools Work

Both tools are single HTML files hosted on GitHub Pages. They connect to Google Sheets via Apps Script web app URLs baked into each file — no login required, no manual data entry. Open the URL, data loads automatically, and you're ready to go.
